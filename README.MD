# ПП.02 — Производственная практика (Автомобильная промышленность)

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по специальности 09.02.07 «Информационные системы и программирование».

Тема практики: **«Автомобильная промышленность»**.

Практика выполнялась на базе **ИП Хусайни А.К.**  
В ходе работы были разработаны программные средства для автоматизации задач автозавода и автосервиса: расчёт стоимости автомобиля, учёт запчастей, запись на сервис, проверка VIN-номеров, подбор автомобилей и генерация госномеров.

Все скрипты работают локально, не требуют подключения к интернету и обеспечивают безопасность обрабатываемых данных.

## Цель работы

Цель производственной практики — получение практических навыков разработки программных средств для автоматизации деятельности предприятия автомобильной промышленности.

Задачи:
- Проанализировать бизнес-процессы автозавода и автосервиса
- Разработать скрипты для автоматизации повседневных задач
- Создать UML-диаграммы для моделирования системы
- Оформить отчётную документацию
- Опубликовать результаты в публичном репозитории
